## 芯笙微纳电子科技上位机项目

### AutoCalibrationTool

自动标定工具，核心功能作为定制化数据处理程序已移植到AutoCommander软件中。

### AutoCommander

是一款界面和指令可灵活自主配置的工具软件现在支持将特定指令和定制化的数据处理逻辑关联，灵活便捷。

### CalibrationTool

MFD/MFM校准工具，是最早开发的MFD/MFM工具软件，现在可以使用AutoCommander替代。

### MFCSoftware

MFC上位机软件，具备数据展示和命令交互的MFC系列上位机控制软件。

### MFCSoftwareForCUP

石油大学上位机项目
